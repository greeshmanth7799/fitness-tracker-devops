# Docker Hub Repository

## Repository Information

Docker Hub repository stores production container images for the Fitness Tracker application.

## Repository URL

https://hub.docker.com/r/greeshmanth7799/fitness-tracker

## Image Tags

* v1
* latest

## Docker Pull Commands

```bash
docker pull YOUR-DOCKERHUB-USERNAME/fitness-tracker:v1
```

```bash
docker pull YOUR-DOCKERHUB-USERNAME/fitness-tracker:latest
```

## Tagging Strategy

| Tag    | Purpose             |
| ------ | ------------------- |
| v1     | Stable release      |
| latest | Most recent release |

## Deployment Usage

These images will be used for:

* Kubernetes deployment
* AWS EKS deployment
* Helm chart deployments
